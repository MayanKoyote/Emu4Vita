# Emu4Vita

## 简介

一个使用[Libretro](https://github.com/libretro) API的Mini前端，目前只支持psvita。

## 支持平台

* Sony Ps Vita

## 如何编译

需要安装好[Vitasdk](https://github.com/vitasdk)，并克隆好项目以及所有子模块。

可在根目录编译所有核心的vpk，也可进入apps文件下的核心文件夹内单独编译,
执行make即可，会自动编译好所需的一切。
